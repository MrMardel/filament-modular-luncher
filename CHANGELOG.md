# Changelog

All notable changes to `filament-modular-luncher` will be documented in this file.

## v1.0.0 - 2026-01-29

### Added
- Initial release of Filament Modular Luncher.
- Module installation via ZIP, URL (Git/ZIP), and Composer.
- Module management (Enable/Disable, Update, Uninstall).
- Backup and Restore functionality.
- Filament Resource integration.
